E-COMMERCE FROM SCRATCH

A simple basic JAVA console project.

This ecommerce project is actually a course, created to teach basic JAVA with no framework usage.

The purpose is to give students an idea on how to create an ecommerce (or any other project) from scratch.

Instead of a course with diferent lessons (1, 2, 3 etc) , each one teaching a certain thing, I decided to put together all separated knowledge in a project.

This way students can finish the course and start coding projects, not only repeat the lessons.

The course does not provide a complete ecommerce project but everytime I got a free time I try to make it better.
